# GraphicEngine
Multiplatformowy silnik graficzny 3D wykonwny w oparciu o OpenGL.

## Struktura katalogów

	/src		Kod źródłowy
	/libs		Biblioteki zewnętrzne
	/docs		Screenshoty
	/test		Testy jednostkowe


## Demo
    ESC - Wyjście z programu
    L-CTRL - Wyłączenie menu
